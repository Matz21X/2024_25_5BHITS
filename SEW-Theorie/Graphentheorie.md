>Ein Graph ist eine Struktur, die aus einer Menge von Knoten (Nodes, Vertices) und einer Menge von Kanten (Edges) besteht:

$$V= [ v1, v2, ..., vn ]$$
$$E= [ e1, e2, ..., en ]$$
